# Stur-vey 설문조사 앱 기능명세서

## 📌 프로젝트 개요

### 프로젝트명
**Stur-vey** (Student Survey)

### 목적
학생들이 필요한 설문조사를 쉽게 생성하고 응답할 수 있는 웹 기반 설문조사 플랫폼

### 기술 스택
- **Frontend & Backend**: Next.js 16 (App Router)
- **Language**: JavaScript
- **Database**: Supabase (PostgreSQL)
- **Authentication**: Supabase Auth (google OAuth)
- **Styling**: CSS Modules
- **Deployment**: Vercel (예정)

---

## 🎯 핵심 기능 (MVP)

### 1. 사용자 인증 및 관리

#### 1.1 회원가입/로그인
- **인증 방식**: 구글 소셜 로그인 전용
- **학생 인증**: 초기에는 별도 인증 없음 (향후 학교 이메일 인증 추가 가능)
- **프로필 정보**:
  - 닉네임 (필수)
  - 학교명 (필수)
  - 프로필 이미지 (구글 연동)
  - 가입일

#### 1.2 사용자 역할
- **일반 사용자**: 설문 생성 및 응답 가능
- **관리자**: 부적절한 설문 삭제/숨김 처리 가능

#### 1.3 마이페이지
- 내가 만든 설문 목록
- 내가 참여한 설문 목록
- 프로필 정보 수정

---

### 2. 설문조사 생성

#### 2.1 설문 기본 정보
- **제목** (필수, 최대 100자)
- **설명** (선택, 최대 500자)
- **카테고리** (필수)
  - 학업/진로
  - 대학생활
  - 취미/관심사
  - 소비/구매
  - 사회/이슈
  - 기타
- **마감 기한** (필수)
- **응답 제한 수** (선택)
- **공개 여부** (공개/비공개)

#### 2.2 질문 유형
지원하는 질문 타입:
1. **객관식 (단일 선택)**
   - 라디오 버튼 형태
   - 최소 2개 이상의 선택지
2. **객관식 (복수 선택)**
   - 체크박스 형태
   - 최소 2개 이상의 선택지
3. **주관식 (단답형)**
   - 한 줄 텍스트 입력
   - 최대 100자
4. **주관식 (장문형)**
   - 여러 줄 텍스트 입력
   - 최대 500자
5. **척도형**
   - 1-5점, 1-10점 선택 가능
   - 시각적 슬라이더 형태

#### 2.3 질문 설정
- **필수/선택 여부** 설정 가능
- **질문 순서** 드래그 앤 드롭으로 변경 가능
- 질문 추가/삭제/수정 자유롭게 가능

#### 2.4 설문 설정 옵션
- **중복 응답 방지**: 생성자가 선택 (1인 1회 / 중복 허용)
- **응답 수정 허용**: 생성자가 선택 (허용 / 불허)
- **결과 공개 설정**: 생성자가 선택
  - 모두에게 공개
  - 응답자에게만 공개
  - 생성자만 확인 가능
- **타겟 설정**: 특정 학교만 응답 가능 (선택사항)

#### 2.5 임시저장
- 작성 중인 설문을 임시저장할 수 있음
- 임시저장된 설문은 마이페이지에서 확인 가능
- 임시저장 상태: 작성 중 / 발행됨

---

### 3. 설문조사 응답

#### 3.1 응답 조건
- **로그인 필수**: 비로그인 사용자는 응답 불가
- 중복 응답 여부는 설문 설정에 따름
- 마감된 설문은 응답 불가

#### 3.2 응답 프로세스
1. 설문 상세 페이지 접근
2. 모든 필수 질문 응답
3. 응답 제출 전 미리보기 화면 제공
4. 최종 제출

#### 3.3 응답 후 기능
- **내 응답 확인**: 제출한 답변 재확인 가능
- **응답 수정**: 설문 설정에 따라 수정 가능/불가
- **결과 확인**: 설문 설정에 따라 전체 통계 확인 가능

---

### 4. 설문 검색 및 탐색

#### 4.1 메인 페이지
설문 정렬 옵션:
- **최신순** (기본)
- **마감 임박순**
- **인기순** (응답 수 기준)

탭 구성:
- 전체 설문
- 카테고리별 설문

#### 4.2 검색 기능
- **제목 검색**: 설문 제목으로 검색
- **내용 검색**: 설문 설명 포함 검색
- **카테고리 필터**: 특정 카테고리만 표시
- **학교 필터**: 특정 학교 설문만 표시

#### 4.3 설문 상태 표시
- **진행 중**: 마감 기한 전, 응답 가능
- **마감됨**: 마감 기한 초과 또는 응답 제한 수 달성
- **응답 완료**: 내가 이미 응답한 설문 (배지 표시)

---

### 5. 결과 분석 및 통계

#### 5.1 설문 생성자 대시보드
확인 가능한 정보:
- **전체 응답 수**
- **질문별 통계**
  - 객관식: 파이 차트, 막대 그래프
  - 주관식: 텍스트 목록
  - 척도형: 평균값, 분포 그래프
- **개별 응답 상세 내용**
  - 응답자별 전체 답변 확인
  - 익명 처리 (닉네임 비공개, 통계적 정보만)

#### 5.2 데이터 내보내기
- **형식**: CSV (Excel 호환)
- **포함 내용**:
  - 질문별 통계 요약
  - 개별 응답 전체 데이터
- **다운로드 버튼**: 결과 페이지에서 바로 다운로드

#### 5.3 결과 업데이트
- 새로고침 시 최신 데이터 반영
- 실시간 업데이트는 MVP에서 제외

---

### 6. 공유 기능

#### 6.1 설문 공유
- **URL 링크 복사**: 클립보드에 자동 복사
- **카카오톡 공유**: 카카오톡 공유하기 SDK 연동
  - 썸네일: 설문 제목, 설명, 마감일 표시
  - 버튼: "설문 참여하기"

---

### 7. 관리자 기능

#### 7.1 설문 관리
- 부적절한 설문 **삭제** 또는 **숨김 처리**
- 관리자 전용 대시보드
  - 최근 생성된 설문 목록
  - 신고된 설문 목록 (향후 추가)

#### 7.2 사용자 관리
- 문제 사용자 계정 정지 (향후 추가)

---

### 8. UI/UX

#### 8.1 디자인 스타일
- **모던하고 깔끔한 디자인**
- **카드형 레이아웃**: 설문 목록
- **직관적인 아이콘 사용**
- **색상 테마**: 포인트 컬러 활용

#### 8.2 반응형 디자인
- **모바일 우선** 설계
- 태블릿, 데스크톱 대응
- 모든 화면 크기에서 최적화된 경험 제공

#### 8.3 접근성
- 시맨틱 HTML 사용
- 키보드 네비게이션 지원
- 명확한 레이블 및 설명

---

### 9. 데이터 보안 및 개인정보

#### 9.1 개인정보 처리
- **응답 데이터**: 통계에만 활용, 개별 식별 불가
- **닉네임**: 결과 페이지에서 비공개
- **프로필 정보**: 최소한의 정보만 수집

#### 9.2 데이터 보관
- **설문 데이터**: 영구 보관
- **설문 삭제**: 응답이 없는 경우에만 가능
- **응답 데이터**: 설문 삭제 시 함께 삭제
